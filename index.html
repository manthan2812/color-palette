Colours are specified using Predefined Colour Names or RGB, HEX, HSL, RGBA, HSLA Values.
An RGB colour value represents Red, Green and Blue Light Sources.
In CSS, a colour can be specified as an RGB value, using this formula: rgb(red, green, blue)
 A Hexadecimal colour is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the colour.
In CSS, a colour can be specified using a Hexadecimal value in the form: #RRGGBB
 An HSL stands for Hue, Saturation and Lightness.
In CSS, a colour can be specified using Hue, Saturation and Lightness (HSL) in the form: hsl(hue, saturation, lightness)
